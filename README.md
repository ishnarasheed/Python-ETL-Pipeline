# ⚡ Python ETL Pipeline

A professional Python-based Extract, Transform, Load (ETL) workflow designed to automate data processing and cleaning.

## 🚀 Overview
This project focuses on building an efficient data pipeline that handles raw datasets, cleans anomalies, and prepares data for analytical consumption.

## 🛠 Tech Stack
- **Language:** Python
- **Libraries:** Pandas

## 📂 Project Structure
- `data/`: Contains source and processed datasets.
- `src/`: Contains the core ETL transformation logic.

## 🔧 Features
- **Extraction:** Reads data from raw CSV files.
- **Transformation:** Normalizes column names and handles missing/null values.
- **Loading:** Exports cleaned, structured data for database ingestion.
